## Tentang Project

Alisa Bakery adalah website toko roti lokal berbasis HTML, CSS, dan JavaScript.  
Project ini dibuat sebagai latihan front-end development dan portofolio website statis dengan sistem pemesanan sederhana yang terintegrasi dengan WhatsApp.

Website ini menampilkan produk roti, halaman order, checkout, serta sistem redirect otomatis ke WhatsApp dengan detail pesanan lengkap.


## Fitur Utama

- Desain modern dan responsive
- Slider produk interaktif
- Halaman detail order
- Halaman checkout
- Redirect otomatis ke WhatsApp dengan format pesan lengkap
- Validasi estimasi kedatangan minimal 15 menit (karena preparation time)
- Deploy menggunakan GitHub Pages


## Alur Pemesanan

1. User memilih produk
2. Mengatur jumlah pesanan
3. Memilih metode pesanan (Dine In / Delivery Coming Soon)
4. Mengisi data pemesan
5. Mengatur estimasi kedatangan (minimal 15 menit)
6. Klik "Beli Sekarang"
7. Sistem redirect ke WhatsApp dengan detail lengkap pesanan
